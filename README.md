## Parte 2
Você encontrará uma engine de processamento de dados em Python 3.8 capaz de
processar todos os dados presentes na `parte_2/routing-keys/`.

Para executar esse projeto é importante instalar as dependências presentes
no `requirements.txt`.

Para avaliar esse exercício, mantenha seu cliente shell na pasta
`desafio_youse/parte_2`

A engine é invocada ao executar

`python main.py <routing-key-path>`

Os testes unitários são executados com `py.test`. Para visualizar os resultados
execute

`pytest`

ou caso queira executar um script em particular

`pytest <path-do-script>`
